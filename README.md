# 🤖 Rule-Based AI Chatbot

> **DecodeLabs AI Internship Programme — Project 1**

A production-quality, rule-based AI chatbot built entirely in Python.
Uses a two-layer dictionary architecture for O(1) intent resolution —
no if-elif chains, no machine learning, pure deterministic logic.

---

## ✨ Features

- 🗣️ 22 intents with 65+ phrase mappings
- ☀️ Time-based greeting (Morning / Afternoon / Evening)
- 🎨 Coloured terminal interface using Colorama
- 📊 Session analytics displayed on exit
- ⚡ O(1) dictionary lookup — zero if-elif chains
- 🛡️ Graceful error handling with try/except/finally
- 🔁 Continuous conversation loop
